About this Site

I hand-coded this site. This site was created using Python Flask and MongoDB. Flask fit well, because I like the fact that is a microframework, and therefor not bloated, but so easy to extend as needed. I chose MongoDB as I only needed a database for the blog posts and comments. That is why I went with a NoSQL dbms.

This site also uses two jQuery plugins: Slicebox: The jQuery 3D image slider by codrops and jquery-easing v1.3 by GSGD
